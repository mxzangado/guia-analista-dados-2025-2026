# 📊 Guia de Carreira: Analista de Dados (2025/2026)

Este repositório é um guia completo para quem deseja se tornar um **Analista de Dados**, com foco em habilidades técnicas, ferramentas, mercado e evolução de carreira.

---

## 🧠 Estrutura do Guia

* 📌 Fundamentos Técnicos
* 📊 Ferramentas de BI e Visualização
* 🔍 Processo de Análise
* 🤖 O Impacto da IA
* 🧩 Soft Skills Essenciais
* 💼 Mercado e Carreira

---

## 🚀 Objetivo

Ajudar iniciantes e profissionais em transição a construir uma base sólida em análise de dados e conquistar oportunidades no mercado.

---

## 🛠️ Tecnologias abordadas

* Python (Pandas, NumPy)
* SQL
* Power BI / Tableau
* Excel
* Estatística básica
* Machine Learning (introdução)
