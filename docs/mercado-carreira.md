# 💼 Mercado de Trabalho e Carreira em Dados

## 🌍 Visão geral

A área de dados está em alta no mundo todo, com grande demanda por profissionais qualificados.

---

## 📊 Principais cargos

* Analista de Dados (Data Analyst)
* Cientista de Dados (Data Scientist)
* Engenheiro de Dados (Data Engineer)
* BI Analyst

---

## 💰 Faixa salarial (média)

* Júnior: R$ 2.500 – R$ 5.000
* Pleno: R$ 5.000 – R$ 10.000
* Sênior: R$ 10.000+

*(Pode variar por país e experiência)*

---

## 🌍 Mercado internacional

* Alta demanda na Europa 🇵🇹
* Empresas valorizam portfólio prático
* Inglês é diferencial importante

---

## 🎯 Como entrar na área

### 1. Aprender fundamentos

* Python
* SQL
* Estatística

### 2. Criar portfólio

* Projetos no GitHub
* Dashboards
* Estudos de caso

### 3. Aplicar para vagas

* LinkedIn
* Sites de emprego
* Networking

---

## 📈 Crescimento na carreira

* Especialização em áreas específicas
* Aprender Machine Learning
* Evoluir para cargos mais estratégicos

---

## 🚀 Dica final

Mais importante que certificados é mostrar na prática o que você sabe fazer.
