# 🧩 Soft Skills para Analista de Dados

## 🧠 O que são

Soft skills são habilidades comportamentais essenciais para trabalhar bem com pessoas e comunicar resultados.

---

## ⭐ Principais soft skills

### 🗣️ Comunicação

* Explicar dados de forma simples
* Apresentar insights para não técnicos

---

### 🧠 Pensamento crítico

* Questionar dados
* Evitar conclusões erradas

---

### 🧩 Resolução de problemas

* Encontrar soluções baseadas em dados
* Identificar causas de problemas

---

### 🤝 Trabalho em equipe

* Colaborar com outras áreas
* Entender necessidades do negócio

---

### ⏳ Gestão de tempo

* Priorizar tarefas
* Cumprir prazos

---

## 📊 Exemplo prático

Transformar uma análise complexa em um dashboard simples para gestores.

---

## 🚀 Como desenvolver

* Praticar apresentações
* Trabalhar em projetos em grupo
* Estudar storytelling com dados

---

## 📌 Dica

Um analista que sabe se comunicar bem vale mais do que um técnico que não consegue explicar seus resultados.
