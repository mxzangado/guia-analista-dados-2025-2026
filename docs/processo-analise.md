# 🔍 Processo de Análise de Dados

## 🧠 O que é

O processo de análise de dados é o conjunto de etapas utilizadas para transformar dados brutos em insights úteis para tomada de decisão.

---

## 🔄 Etapas do Processo

### 1. 📥 Coleta de Dados

* APIs
* Bancos de dados
* Planilhas (Excel, CSV)

---

### 2. 🧹 Limpeza de Dados

* Remover valores nulos
* Tratar dados inconsistentes
* Padronizar formatos

---

### 3. 📊 Análise Exploratória (EDA)

* Estatísticas descritivas
* Identificação de padrões
* Visualização inicial dos dados

---

### 4. 📈 Modelagem (opcional)

* Regressão
* Classificação
* Machine Learning básico

---

### 5. 📊 Visualização e Comunicação

* Dashboards
* Gráficos
* Storytelling com dados

---

## 🛠️ Ferramentas usadas

* Python (Pandas, Matplotlib, Seaborn)
* SQL
* Power BI / Tableau
* Excel

---

## 📊 Exemplo prático

Analisar vendas de uma empresa:

* Identificar produtos mais vendidos
* Detectar sazonalidade
* Melhorar estratégia de vendas

---

## 🚀 Como aprender

* Praticar com datasets públicos (Kaggle)
* Criar projetos reais
* Fazer análises completas do início ao fim
